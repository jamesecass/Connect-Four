# Connect-Four

A Python implementation of the classic Connect Four game with win detection logic.

---

## 🚀 Overview

This project recreates Connect Four using Python, focusing on grid-based logic and algorithmic win detection.

The game simulates:
- 2-player gameplay
- Vertical, horizontal, and diagonal win detection
- Board updates after each move

---

## 🧠 Features

- 6x7 grid system
- Token drop physics logic
- Full win-condition detection
- Clean and readable structure

---

## 🛠 Tech Stack

- Python
